1. Buka folder project di VS Code

## ğŸ”„ Migrasi ke CMS/Backend

### JSON ke API

**Sekarang:**
```javascript
const data = await loadData('../data/home.json');
```

**Nanti (API):**
```javascript
const data = await loadData('/api/home');
```

### Partials ke PHP

**Sekarang:** JavaScript renders navbar/footer  
**Nanti:** Replace dengan PHP includes

```php
<?php include 'partials/navbar.php'; ?>
```

### Environment Config

Buat `.env` file:
```env
API_BASE_URL=https://api.eventprint.com
```

Update `utils.js`:
```javascript
const API_URL = process.env.API_BASE_URL || '';
```
